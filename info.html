Tasca 9.2. Cerca de fotografies amb AJAX a Flickr en format JSON

Objetivo de la práctica

El objetivo de esta práctica es realizar una página web que muestre fotografías de Flickr. La información de las fotografías debe mostrarse de manera dinámica y dibujarse en una tabla.

Uso de la aplicación

Las imágenes son buscadas en Flickr según las etiquetas o "tags" introducidas en la barra de búsqueda. Se puede seleccionar búsqueda de fotografías que contengan todas o alguna de las etiquetas.

Explicación técnica

La llamada a la página Flickr se realiza mediante el método getJSON (AJAX) de JQuery. Se modifican los dos parámetros "tags" y "tagmode" para obtener las imágenes con todas o alguna de las "tags" introducidas pr el usuario.
Cuando la llamada devuelve el JSON, se recorre la información de las imágenes, dibujando en una tabla de Bootstrap la información (título de la foto, miniatura, fecha, autor, etiquetas y enlace a la imagen original). Cada vez que se hace la consulta, la tabla resultante se redibuja completamente.
